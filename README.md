# Phonebook 
### Веб-приложение Телефонная книга
Инструменты:
* Java
* Spring Boot
* Spring MVC
* Spring Data JDBC
* PostgreSQL
* HTML
* CSS
* Thymeleaf
* Log4j
* JUnit 5
